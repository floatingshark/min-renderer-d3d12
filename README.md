A minimum-scale rendering application with d3d12.  
This project can be built on vscode.  

<p align="left">
	<img src="./ScreenShot/overview.png" width="60%">

## Rendering Features
Forward Rendering  
MSAA  
Shadow Mapping  

## Requirements to compile project
Clang  
CMake version >=3.5  
Ninja  
MSVC x86-64 build  
Windows SDK
