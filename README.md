A simple rendering application with DirectX12 and ~~Vulkan~~ (not yet)  
This project is configured by clang + GLFW + DirectX. You can build this repository on vscode.  

<p align="center">
	<img src="./ScreenShot/overview.png" width="100%">
</p>

## Features
Forward Rendering  
MSAA  
Shadow Mapping  

## Requirements

- Clang/LLVM
- CMake version >=3.5  
- Ninja

DirectX12
- MSVC x86-64 build  
- Windows SDK
